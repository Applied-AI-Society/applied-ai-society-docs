## Usage Rules

### Do

- ✅ Use official assets from this page
- ✅ Maintain clear space around logos
- ✅ Use appropriate color version for background
- ✅ Keep logos at readable sizes

### Don't

- ❌ Recreate logos from scratch
- ❌ Change logo colors beyond light/dark versions
- ❌ Stretch or distort logos
- ❌ Add effects (shadows, gradients, outlines)

---

## File Formats

| Format | Use |
|--------|-----|
| **SVG** | Web, digital, scalable graphics |
| **PNG** | Social media, presentations (when SVG not supported) |
| **PDF** | Print materials |

All logos are provided in SVG format. If you need PNG or PDF versions, export from the SVG at the size you need.
